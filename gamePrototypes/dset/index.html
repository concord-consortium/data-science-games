DSET Connecting Technology Sessions

Before the Session

Be sure to...

  • …bring a laptop!
  • …make sure you have a web server installed that can serve local files on your machine.
    Your computer may have this already. Check out the Getting Started document.
  • …download the starter files (the dset folder) to your server's root directory.
    You will find a set of these files here.

Session 1: Thursday 9:00–10:15

In this session, you'll program a CODAP plugin—an estimation game—in JavaScript, or possibly NetLogo. Click here for instructions.

For the "vanilla" Mac situation, if everything is installed properly, you can run it with this URL:
http://codap.concord.org/releases/dsg?di=http://localhost/dset/estim8/estim8.html

If you have changed your server or are not on a Mac, alter the above URL to point to the dset folder.

Session 2: Thursday 1:45–3:00

In this session, you'll teach your plugin to repond to CODAP events such as case selection.
